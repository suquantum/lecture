# Machine Learning 101
This is a hands on material for machine learning 101. This mateiral focus on introduction of machine learning topic for those not familiar with data science, programing, mathematics, etc. to get image on what machine learning is.
The prerequiseites are the following.

1. Understanding of basic probability
2. Data type such as integer, float, string.
3. Be able to set up environements.

For the #3, if this is challenging, please contact the author for revise the set-up manual.

# Installing to Local Computer
Run the following command at the root folder.
```bash
conda env create -f ml_101.yml
```
Then activate the environment with the following command.
```bash
conda activate ml_101
```
You will be able to run jupyter notebook with
```
jupyter notebook
```
or
```
jupyter lab
```
